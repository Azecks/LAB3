## Задача 1

### Текст задачи

На основе последовательности вещественных чисел получить последовательность из их последних цифр.

### Алгоритм решения

1)Получить последовательность чисел
2)Для каждого числа в последовательности:
	-Преобразовать число в строку 
	-Найти последний символ строки
	-Преобразовать символ в число
3)Вернуть новую последовательность последних цифр

### Текст задачи

Последовательность содержит символы. Составить из них строку

### Алгоритм решения

1)Получить последовательность символов
2)Начать с пустой строки "" как начальное значение acc
3)Для каждого символа ch в последовательности:
	-Добавить символ к накопленной строке (acc + string ch)
4)Вернуть полученную строку
